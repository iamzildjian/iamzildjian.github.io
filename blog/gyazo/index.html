In this tutorial, I will show you how to use and bypass the recording time limit of Gyazo free version. As we all know, in free version of gyazo you can only take no longer than 7 seconds of video recording. So read the full article to learn how to bypass the recording limit of gyazo.

Installer

  • Gyazo

  • dnSpy v.6.1.8

Guide


Read the instructions carefully！


  1. First of all, gyazo is a great tool where you can screenshot, record video and convert it to GIF. You can also share it easily in just few clicks. Visit this site to learn more how to use gyazo. Download the free version of gyazo and dnSpy (64 bit) provided above.


  2. Install Gyazo.


  3. After installing gyazo, extract dnSpy and run dnSpy.exe as Administrator.


  4. Next you need to drag and drop GyazoGIF.exe located at "C:\Program Files (x86)\Gyazo" inside the dnSpy.


  5. Inside dnSpy, click GyazoGIF > GyazoGIF.exe > Gyazo.GyazoGIF.Model>.Capture > MaxRecordingTimeService and look for MaxRecordingTimeForLongRecording.


  6. After clicking MaxRecordingTimeForLongRecording, right click on it and choose edit method (C#). Change the values of these 2 strings "MaxRecordingTimeService.MaxRecordingTimeForLongRecording = TimeSpan.FromSeconds(180.0)" and "MaxRecordingTimeService.MaxRecordingTimeForShortRecording = TimeSpan.FromSeconds(7.0)" to any numbers that you like then click compile.


  7. After compiling, click File then Save Module and save it to any destination folder that you want.


  8. Go to the folder where you save the compiled files then copy & merge the files to the default installation folder of gyazo.


  9. Now try to use Gyazo GIF recording (ctlr+shift+G) if you can take more than 7 seconds.
